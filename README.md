# Advanced Python Calculator Package

This project is a reusable Python calculator package created for learning GitHub collaboration.

## Features

The package supports:

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Floor Division
- Powers
- Square Roots
- Factorials
- Percentages
- Trigonometry
- Mean
- Median
- Mode
- Minimum
- Maximum
- Range
- Calculation history
